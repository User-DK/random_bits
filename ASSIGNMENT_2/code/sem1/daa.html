About the Course

This course aims to illustrate and apply algorithm analysis techniques, discuss efficient algorithms for various problems, and explain and demonstrate different algorithm techniques for real-world problems. It also focuses on computing and proving the complexity class of various algorithm techniques.

Course Code: 6CS302

Discrete Mathematics, Data Structure

Course Objectives

  • To illustrate and apply the algorithm analysis techniques.
  • To discuss the efficient algorithm for various problems.
  • To explain and demonstrate different algorithm techniques for real-world problems.
  • To compute and prove the complexity class of various algorithm techniques.

Course Outline

  • Module 1: Introduction to Algorithm
  • Module 2: Divide and Conquer Method
  • Module 3: Greedy Method
  • Module 4: Dynamic Programming Method
  • Module 5: Backtracking & Branch and Bound Method
  • odule 6: Class of Problem & Parallel Algorithms
