About the Course

This course aims to introduce the fundamentals of compiler design and the various tools used in designing a compiler. It covers the role of different phases involved in the design of a compiler and provides an in-depth understanding of each phase. The course also focuses on the practical aspects of designing various phases of a compiler using compiler design tools and techniques.

Course Code: 6CS301

Desired Requisites: Formal Language and Automata Theory, Discrete Mathematics

Course Objectives

  • To introduce fundamentals of compiler design and various tools used to design a compiler.
  • To inculcate role of various phases involved during design of a compiler and impart in-depth working of each phase.
  • To exercise design of various phases of a compiler using compiler design tools and techniques.

Course Outline

  • Module 1: Fundamentals of Compiler
  • Module 2: Syntax Analysis
  • Module 3: Syntax Directed Translation & Run time environments
  • Module 4: Intermediate Code Generation
  • Module 5: Code Optimization
  • Module 6: Code Generation
