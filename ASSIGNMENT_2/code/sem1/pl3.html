About the Course

This course aims to inculcate an understanding of the World Wide Web, Internet, the concepts of web applications development, and web programming languages. It also introduces state-of-the-art front-end and back-end development frameworks of web programming.

Course Code: 6CS352

Desired Requisites: Basics of Object-Oriented Programming

Course Objectives

  • To inculcate understanding of the World Wide Web, Internet, and the concepts of web applications development.
  • To introduce the selection of appropriate concepts of internet and web programming such as HTML, CSS, JavaScript, and other server-side scripting languages.
  • To introduce the selection of appropriate concepts from different state-of-the-art frameworks/libraries and tools for developing a web application.
  • To infuse skills of combining different components from state-of-the-art technologies to design a web and mobile app to solve real-world problems.

Course Outline

  • Module 1: Introduction to World Wide Web
  • Module 2: Client-side Programming
  • Module 3: Server-side Programming
  • Module 4: Web Application Framework
  • Module 5: Server-side Development Framework
  • Module 6: Hosting Web Applications
