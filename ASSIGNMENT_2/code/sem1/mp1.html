About the Course

This course aims to inculcate an understanding of the World Wide Web, Internet, the concepts of web applications development, and web programming languages. It also introduces state-of-the-art front-end and back-end development frameworks of web programming.

Course Code: 6CS341

Desired Requisites: Nil

Course Objectives

  • To provide hands-on experience in developing a small-scale software project.
  • To undergo project management techniques and project design principles.
  • To implement the project with appropriate programming languages and testing tools.
  • To develop analytical vision and skills to analyse, compare the outcome with other techniques.
